# Account  
This is an object representing a user account. You can create new user, update and delete current account info.
Account login through email only.

## Login Email
Endpoint:
```perl
GET /account/login
```
<br/>
Expected Body:
```JSON
"account":{
	"email": "johnbeck@email.com",
	"password": "passwordone"
}
```
